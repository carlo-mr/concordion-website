Extensions

This page lists some extensions that add features to Concordion. See the Extensions API for details of how to create extensions.

Alternative Input Sources

  • Excel Extension — enables Concordion to use an Excel spreadsheet as an input specification

Specification Enhancers

  • Screenshot Extension — add screenshots to Concordion output to diagnose problems or improve the documentation
  • Logging Tooltip Extension — add logging information as tooltips to the Concordion output
  • Timestamp Formatter Extension — change the footer of the Concordion output to show hours, minutes and seconds
  • Translator Extension — modify the text of exception messages

Additional Commands

  • Screenshot Extension — add screenshots to Concordion output to diagnose problems or improve the documentation
  • Embed Extension — embed HTML in the Concordion output

Notes

The combined concordion-extensions project has been deprecated, and split into the separate projects above.

Additional contributions are welcome. Please get in touch on the concordion-dev list.
