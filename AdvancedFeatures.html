Advanced Features

Attributes

Concordion.NET does not implement the annotations / attributes of the Java version: "@ExpectedToPass", "@ExpectedToFail", and "@Unimplemented".

Instead the Concordion-NUnit-Addin supports using NUnit attributes on Concordion.NET fixture classes. Thus, you can use the IgnoreAttribute to achieve a similar behavior as the @Unimplemented annotation:

            using Concordion.Integration;
            namespace Kickstart.Spec.HelloWorld
            {
                [ConcordionTest]
                [Ignore("ignore fixture and active specification")]
                public class HelloWorldTest
                {
                    public string GetGreeting()
                    {
                        //add call to your system under test here
                    }
                } 
            }
